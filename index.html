Project_Overview

In this challenge project plotly and javaScript knowledge is going to be used to create charts to show belly_button Biodiversity using Data. This project will be helpful for researchers to visualize the data to know about biodiversity. It can also be used by anyone who want to save him/herself from any infection. Select the Test Subject ID number from the dropdown menu. The demographic information table and the following charts will update to display the individual's data. The horizontal bar chart identifys the top 10 bacterial species in their navel. The bubble chart displays bacteria culture test results for each sample. The gauge chart reveals The Washing Frequency.
